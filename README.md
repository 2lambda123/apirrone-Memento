# Poor man's rewind.ai

A simple reimplementation or rewind.ai using chromadb

## Install

```console
$ pip install -e .
```

## Usage
### Background process
Run in a terminal  
```console
$ pmr-bg
```

### Query :

```console
$ pmr-query
```

### Show timeline : 

```console
$ pmr-timeline
```
