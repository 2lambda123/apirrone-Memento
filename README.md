# Poor man's rewind.ai

A simple reimplementation or rewind.ai using chromadb

## Install

```console
$ pip install -e .
```

## Usage
### Background process
Run in a terminal  
```console
$ pmr-bg
```

### Timeline mode: 

```console
$ pmr-timeline
```

Then use `ctrl+f` to search
